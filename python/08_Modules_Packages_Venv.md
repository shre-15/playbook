# Modules, Packages, Virtual Environment

## Module

Python file.

```python
math_utils.py
```

Import:

```python
import math_utils
```

---

## Package

Collection of modules.

```text
mypackage/
    __init__.py
    module1.py
```

---

# Virtual Environment

Create:

```bash
python -m venv myenv
```

Activate:

Windows:

```bash
myenv\Scripts\activate
```

Linux/Mac:

```bash
source myenv/bin/activate
```

---

# Interview Questions

## Why use virtual environment?

Dependency isolation.

---

## Difference between module and package?

Module = single file

Package = collection of modules
