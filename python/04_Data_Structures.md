# Data Structures

# Lists

Ordered and mutable.

```python
numbers = [1,2,3]
```

Methods:

```python
append()
extend()
remove()
pop()
sort()
reverse()
```

---

# Tuples

Ordered and immutable.

```python
data = (1,2,3)
```

---

# Dictionaries

Key-value pairs.

```python
student = {
    "name":"John",
    "age":20
}
```

Methods:

```python
keys()
values()
items()
get()
update()
```

---

# Sets

Unordered unique values.

```python
nums = {1,2,3}
```

Methods:

```python
add()
remove()
union()
intersection()
```

---

# Interview Questions

## List vs Tuple

| List | Tuple |
|--------|--------|
| Mutable | Immutable |
| [] | () |
| Slower | Faster |

---

## Dictionary vs Set

Dictionary:

```python
{
"name":"John"
}
```

Set:

```python
{
1,2,3
}
```
